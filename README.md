# FeelBetters 😊

FeelBetters is a simple Java console application that provides motivational and emotional support quotes based on the user's current mood.

## Features

* Detects user mood input
* Displays random quotes
* Supports multiple moods:

  * Happy 😊
  * Sad 😢
  * Angry 😡
  * Motivate 🔥
* Beginner-friendly Java project
* Uses `Scanner`, `Random`, and `if-else` statements

## Technologies Used

* Java
* Console Application

## How to Run

### Compile the Program

```bash
javac FeelBetters.java
```

### Run the Program

```bash
java FeelBetters
```

## Example

```text
Enter Your Current Mood: happy

😊 Happiness grows when shared with others.
```
